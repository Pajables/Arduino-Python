.. _command_devices:

Command Devices Package
************************
This is the documentation for the Arduino devices that can be controlled with the Commanduino library.
For more information on Axis control, please see the :ref:`devices`.
Here is a list of all supported devices:

    * :ref:`cmdAR`
    * :ref:`cmdAW`
    * :ref:`cmdDev`
    * :ref:`cmdDR`
    * :ref:`cmdDW`
    * :ref:`cmdLAS`
    * :ref:`cmdServ`

For registering each device, please see the :ref:`cmdReg`
